Predictive Maintenance

Equipment Failure Prediction Dashboard

LSTM NASA Dataset Grafana Time-Series

The Problem

Unplanned equipment failures in energy operations cost millions in downtime and emergency repairs. Traditional maintenance schedules are either too frequent (wasting resources) or too infrequent (risking failures). Operators need to predict failures before they happen.

The Solution

This dashboard uses LSTM neural networks trained on NASA's C-MAPSS turbofan engine degradation dataset to predict Remaining Useful Life (RUL) of industrial equipment. The model analyzes sensor time-series data to forecast when maintenance is needed.

Grafana dashboards provide real-time visualization of equipment health scores, enabling maintenance teams to prioritize interventions and optimize schedules.

Architecture

  • Data Source: NASA C-MAPSS (Commercial Modular Aero-Propulsion System Simulation)
  • Model: LSTM with attention mechanism for sequence modeling
  • Features: 21 sensor readings + operational settings
  • Visualization: Grafana dashboards with real-time alerts
  • Deployment: Azure ML for training, FastAPI for inference

Technologies

  • TensorFlow / Keras (LSTM)
  • NASA C-MAPSS Dataset
  • Grafana

  • Python (FastAPI, Pandas, NumPy)
