Geo-Insight HSE

Real-time PPE Detection for HSE Compliance

YOLOv8 Azure Container Apps Edge AI Computer Vision

The Problem

Health, Safety, and Environment (HSE) is the #1 priority in energy operations. Manual PPE compliance monitoring is slow, inconsistent, and impossible to scale across large industrial sites. Safety violations often go undetected until incidents occur.

The Solution

Geo-Insight HSE is a real-time computer vision pipeline that automatically detects PPE violations (helmets, vests, safety glasses) using YOLOv8 deployed on Azure Edge devices. The system processes live camera feeds and sends instant alerts to supervisors.

Built for air-gapped OT environments, the model runs locally on edge devices with sub-100ms latency, ensuring compliance even without internet connectivity.

Architecture

  • Model: YOLOv8 fine-tuned on industrial PPE dataset
  • Edge Runtime: ONNX optimized for Azure Stack Edge / Jetson Nano
  • Data Flow: Camera → Edge Device → OPC UA → SCADA/HMI
  • Alerts: Real-time notifications to supervisors via Teams/SMS
  • Compliance: Designed for Purdue Model Level 3 (air-gapped)

Technologies

  • YOLOv8 (Ultralytics)
  • Azure Container Apps / Azure Stack Edge
  • ONNX Runtime
  • Python (FastAPI)
  • OPC UA Protocol
  • Docker / Kubernetes
