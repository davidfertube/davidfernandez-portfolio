Chat-with-Assets RAG

Enterprise Knowledge Retrieval for Technical Documentation

Azure AI Search Next.js RAG Pipeline Gemini 3 Flash

The Problem

Field operators in the energy sector spend hours searching through hundreds of pages of technical drilling manuals, P&IDs, and equipment specifications. Critical information is buried in PDFs, and institutional knowledge walks out the door when experienced engineers retire.

The Solution

Chat-with-Assets is a Retrieval-Augmented Generation (RAG) pipeline that indexes 500+ pages of technical documentation and enables natural language queries. Field operators can ask questions like "What's the max pressure for valve V-2045?" and get instant, accurate answers with source citations.

Built on Azure AI Search with Pinecone vector storage and Gemini 3 Flash for generation, the system delivers sub-200ms responses while maintaining enterprise security and compliance requirements.

Architecture

  • Document Processing: Unstructured.io for multi-column PDF parsing
  • Embeddings: Azure OpenAI text-embedding-ada-002
  • Vector Store: Pinecone with hybrid search (dense + sparse)
  • LLM: Gemini 3 Flash with structured output
  • Frontend: Next.js with streaming responses
  • Security: Azure AD authentication, role-based access
